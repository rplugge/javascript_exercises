If you collect the user's input as an integer (i.e. by using parseInt() to turn it into an integer), can you use == to compare? ===? Which is better in this case?

You can still use '==' and '==='. It would be better to use '===' if you're parsing the variable into an integer.

What is the difference between == and ===?

'==' checks if the values are equal (regardless or integer or string), '===' checks if they are equal and that their class is equal (both strings or integers).
